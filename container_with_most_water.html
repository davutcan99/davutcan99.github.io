LeetCode Problem: Container With Most Water

Problem Description

The problem "Container With Most Water" presents a scenario where there are n vertical lines drawn on a graph. The height of each vertical line is given in an integer array called "height." The two endpoints of each line are represented by coordinates (i, 0) and (i, height[i]). The task is to find two lines that, together with the x-axis, form a container such that it contains the most water.

The goal is to determine the maximum amount of water that the container can store. It is important to note that the container cannot be slanted, and the water level should align with the x-axis.

Let's consider an example to understand the problem requirements:

Example 1:

Input: height = [1, 8, 6, 2, 5, 4, 8, 3, 7]

Output: 49

Explanation: The vertical lines are represented by the array [1, 8, 6, 2, 5, 4, 8, 3, 7]. In this case, the maximum area of water (blue section) that the container can contain is 49.

Approach

To solve this problem, we can use a brute-force approach that considers every possible pair of lines and calculates the area of the container formed by the lines and the x-axis. We then keep track of the maximum area encountered so far and return it as the result.

Here's the algorithm:

  1. Initialize a variable max to keep track of the maximum area encountered.
  2. Iterate through each line of the array, starting from the first line.
    • Initialize a variable count to 1, representing the width of the container.
    • Initialize a variable zahl to 0, representing the calculated area of the container.
    • Iterate through the remaining lines, starting from the line after the current line.
      • If the height of the current line is greater than the height of the initial line, calculate the area of the container using the height of the initial line and the width given by count.
      • If the height of the current line is less than or equal to the height of the initial line, calculate the area of the container using the height of the current line and the width given by count.
      • If the calculated area zahl is greater than or equal to the current maximum area max, update max with the new maximum area.
  3. After iterating through all possible pairs of lines, return max as the maximum amount of water the container can store.

Solution

Based on the problem requirements and the approach outlined above, we can implement the following solution:


class Solution {
    public int maxArea(int[] height) {
        int max = 0;
        int count;
        for (int i = 0; i < height.length; i++) {
            count = 1;
            int zahl = 0;
            for (int j = i + 1; j < height.length; j++) {
                if (height[i] < height[j]) {
                    zahl = count * height[i];
                    count++;
                } else if (height[i] >= height[j]) {
                    zahl = count * height[j];
                    count++;
                }
                if (zahl >= max) {
                    max = zahl;
                }
            }
        }
        return max;
    }
}
