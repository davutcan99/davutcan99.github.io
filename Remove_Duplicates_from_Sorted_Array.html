LeetCode Problem: Remove Duplicates from Sorted Array

Problem Description

The problem "Remove Duplicates from Sorted Array" asks us to remove duplicate elements from a given sorted array in-place. The goal is to modify the array so that each unique element appears only once, and return the number of unique elements. The relative order of the elements should be maintained.

To clarify the requirements, let's consider an example. Given an input array nums = [1, 1, 2], the expected output is 2, and the modified array nums should be [1, 2, _], where the underscore represents any value.

Approach

To solve this problem, we can use a two-pointer technique. We'll maintain two pointers: one for the current unique element and another for iterating through the array.

Here's the algorithm:

  1. Initialize a variable k to keep track of the unique elements.
  2. Iterate through the array starting from the second element.
  3. Compare the current element with the previous element.
    • If they are equal, it means we have encountered a duplicate. Skip this element.
    • If they are different, it means we have found a new unique element. Increment k and assign the current element to the k-th position.
  4. Return k, which represents the number of unique elements.

Let's implement the solution in Java:


    class Solution {
        public int removeDuplicates(int[] nums) {
            int k = 1; // Number of unique elements
            for (int i = 1; i < nums.length; i++) {
                if (nums[i] != nums[i - 1]) {
                    nums[k] = nums[i];
                    k++;
                }
            }
            return k;
        }
    }

Custom Judge

To verify the correctness of our solution, a custom judge will test our implementation. The judge will compare the modified array nums with an expected answer array expectedNums of the correct length.

Here's the code that the judge will run:


    int[] nums = [...]; // Input array
    int[] expectedNums = [...]; // The expected answer with correct length
    
    int k = removeDuplicates(nums); // Calls your implementation
    
    assert k == expectedNums.length;
    for (int i = 0; i < k; i++) {
        assert nums[i] == expectedNums[i];
    }

If all assertions pass, it indicates that our solution is correct and will be accepted.

Example

Let's take a look at an example to further understand the problem:

Input:

nums = [1, 1, 2]

Output:

2, nums = [1, 2, _]

The function should return 2, with the first two elements of nums being 1 and 2, respectively. The remaining element is not important (hence represented by an underscore).

Constraints

  • 1 ≤ nums.length ≤ 3 * 10^4
  • -100 ≤ nums[i] ≤ 100
  • nums is sorted in non-decreasing order.
