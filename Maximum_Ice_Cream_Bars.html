LeetCode Problem: Maximum Ice Cream Bars

Problem Description

The problem "Maximum Ice Cream Bars" presents a scenario where a boy wants to buy ice cream bars from a store. The store offers n ice cream bars, and the prices of the ice cream bars are given in an array called "costs." The boy has a certain number of coins to spend and wants to buy as many ice cream bars as possible within the given budget.

The task is to determine the maximum number of ice cream bars the boy can buy with the available coins. It is important to note that the boy can buy the ice cream bars in any order.

Let's consider an example to understand the problem requirements:

Example 1:

Input: costs = [1, 3, 2, 4, 1], coins = 7

Output: 4

Explanation: The boy can buy ice cream bars at indices 0, 1, 2, and 4 for a total price of 1 + 3 + 2 + 1 = 7.

Approach

To solve this problem, we can use the counting sort algorithm. The idea is to sort the ice cream bar prices in ascending order using counting sort. Then, starting from the lowest price, we can iteratively buy ice cream bars until we run out of coins or reach a price that exceeds the available coins.

Here's the algorithm:

  1. Initialize a variable sum to keep track of the total cost and a variable result to count the number of ice cream bars bought.
  2. Sort the array costs in ascending order.
  3. Iterate through the sorted array, starting from the lowest price.
    • Add the current price to sum.
    • If the available coins (coins) minus sum is still non-negative, increment result (indicating that an ice cream bar is bought).
    • If the available coins minus sum becomes negative, it means the boy cannot afford the current price. In this case, return result as the maximum number of ice cream bars that can be bought.
  4. If the entire array is traversed and no return statement is encountered, it means the boy can buy all the ice cream bars within the available coins. Return result as the maximum number of ice cream bars that can be bought.

Let's implement the solution in Java:


    class Solution {
        public int maxIceCream(int[] costs, int coins) {
             int sum = 0;
             int result = 0;
             Arrays.sort(costs);
             for (int i = 0; i < costs.length; i++) {
                 sum += costs[i];
                 if (coins - sum >= 0) {
                     result++;
                 } else {
                     return result;
                 }
             }
             return result;
        }
    }
