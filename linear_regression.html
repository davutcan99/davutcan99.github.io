Linear Regression with Gradient Descent

Introduction

In this blog post, we'll explore the concept of linear regression and its implementation using gradient descent. Linear regression is a powerful technique for modeling the relationship between variables and making predictions. Gradient descent is an optimization algorithm commonly used to train machine learning models, including linear regression.

Code Implementation

Let's take a look at the Python code that implements linear regression with gradient descent:

import numpy as np
    import csv
    import argparse
    
    def gradient_descent(daten):
        # Read data from CSV file
        a = daten[0]
        file = open(a)
        csvreader = csv.reader(file)
        rows = []
        for row in csvreader:
            rows.append(row)
        file.close()
    
        # Initialize variables
        w0_aktuell = float(0)
        w1_aktuell = float(0)
        w2_aktuell = float(0)
        learning_rate = daten[1]
        threshold = daten[2]
        n = 1000
        i = 0
        sse = 0
        sse_son = -1
        iteration = 0
    
        # Perform gradient descent
        while abs(sse - sse_son) > threshold:
            sse_son = sse
            sse = 0
            x = w0_aktuell
            y = w1_aktuell
            z = w2_aktuell
    
            while i < n:
                x1 = float(rows[i][0])
                x2 = float(rows[i][1])
                target_value = float(rows[i][2])
    
                function = float(1 * x + x1 * y + x2 * z)
    
                gradient0 = (target_value - function) * 1
                gradient1 = (target_value - function) * x1
                gradient2 = (target_value - function) * x2
    
                w0_aktuell = w0_aktuell + learning_rate * gradient0
                w1_aktuell = w1_aktuell + learning_rate * gradient1
                w2_aktuell = w2_aktuell + learning_rate * gradient2
    
                sse += pow((target_value - function), 2)
                i += 1
    
            print("{},{},{},{},{}".format(iteration, x, y, z, sse))
            i = 0
            iteration += 1
    
    def main():
        parser = argparse.ArgumentParser()
        parser.add_argument("--data", type=str)
        parser.add_argument("-e", "--eta", type=float, default=0.00005)
        parser.add_argument("-t", "--threshold", type=float, default=0.0001)
        args = parser.parse_args()
        Daten=(args.data,args.eta,args.threshold)
    
        gradient_descent(Daten)
    
    if __name__ == "__main__":
        main()

Explanation

The code starts by importing the necessary libraries and defining the gradient_descent function. The function takes the input data, learning rate, and threshold as parameters.

Inside the gradient_descent function, the data is read from a CSV file. The algorithm then initializes the weights and other variables required for gradient descent. It defines the number of iterations and sets the initial sum of squared errors (SSE) values.

The code enters a while loop that continues until the difference between the current SSE and the previous SSE is less than the threshold. Inside the loop, it performs gradient descent by updating the weights based on the gradients and learning rate.

The inner loop iterates over the data points and calculates the predicted value using the current weights. It calculates the gradients for each weight and updates them accordingly. It also calculates the SSE for each iteration.

Finally, the code prints the iteration number, current weight values, and SSE at each iteration.

Usage

To use the script, follow these steps:

  1. Create a CSV file containing your data, where each row represents a data point with the input features and target value.
  2. Run the script from the command line with the following arguments:
python linear_regression.py --data <path_to_data> --eta <learning_rate> --threshold <threshold_value>

The --data argument specifies the path to your data file, the --eta argument sets the learning rate (default is 0.00005), and the --threshold argument sets the convergence threshold (default is 0.0001).

Conclusion

Linear regression with gradient descent is a powerful technique for modeling and predicting relationships between variables. By iteratively updating the weights based on the gradients, the algorithm finds the best-fit line that minimizes the sum of squared errors. This allows us to make predictions based on new input data.
